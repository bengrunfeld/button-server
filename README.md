# Button Server

Server for the button game.

## To run locally

-   Clone site with `git clone`
-   `cd button-server`
-   `npm install`
-   `npm run now`


## Database access

In the project root, create a `.env` file and fill in the values. E.g.

```
MYSQL_HOST="hostUri"
MYSQL_USER="username"
MYSQL_PASSWORD="password"
MYSQL_DATABASE="dbName"
```
